pclose = Kopen·E·trad·c/r²·(trad/tref)²。径向厚度按开口环截面弯曲刚度近似形成 t³ 影响；厚度越大，低压力越不容易撑开并关闭径向间隙。Hring 不直接线性乘入 pclose，但会通过 sealRatio=(x/Hring)^1.35 和 Agap=kgap·Hring·g·sealRatio 影响起压速度和端隙泄流。图示 Ring 开口为斜切/圆角/局部遮挡，不是垂直贯通矩形缝，因此 kgap 应理解为投影到最窄通道后的等效面积系数，不能直接取全开 1.0。低压段用 ΔP·sealRatio/pclose 判断 Ring 能否撑开；撑不开时保留残余径向间隙 ceff=c·(1−expandRatio)，并按该间隙计算未贴壁泄漏。Kopen 越小，Ring 越容易在低速胀开。

尺寸敏感度： trad 主要改变 pclose（约 t³），Hring 主要改变端隙泄流面积和密封建立长度；随后再看 kgap / g0 / kleak → leakFull → Kopen → kexpand。
标定顺序： ① 用恒速示功图实测 HRS 峰值力反推总等效泄流面积 Atotal； ② 实测 Ring 自由端隙 g0 代入； ③ 用低速（0.1～0.3 m/s）与高速（0.8～1.0 m/s）曲线对比，依次标定 kleak、kgap、leakFull； ④ 用浮动环 FEA 得到 pclose、expandRatio(ΔP)、接触/贴壁状态、开口闭合状态和环向应力，并反算 Kopen、kexpand； ⑤ 最后用 1.5mm 与 2.0mm 壁厚起效速度差，对比实测示功图和 FEA 起效压力，反标定 Kopen。
示功图 + FEA 结合： 示功图用于反推 Atotal 和各速度下的等效泄流；FEA 用于约束 Ring 撑开、贴壁、开口是否闭合，避免 kleak、kgap、leakFull、Kopen 相互补偿出不合理参数。

计算过程说明
① 位移→进入深度：导向座配合长度 Lfit 为进入套管内部的部分；套筒入口到导向座距离 Lentry=Ltube−Lfit，HRS开始进入位移 son=dmid+S/2−Lentry，进入深度 x = max(0, s−son)。
② 密封比：sealRatio = (x/Hring)^1.35，x=0 时 sealRatio=0 → 附加力强制为0。
③ 流量：Q = Aeff × v；建压有效流量 Qin = Q·sealRatio（m³/s）。
④ 槽面积（圆弧段）：d(x) = d₀·(1−x/L)^m，A_seg = R²·arccos[(R−d)/R] − (R−d)·√(2Rd−d²)，A_groove = N·A_seg。
⑤ 低压撑开：pclose = Kopen·E·trad·c/r²·(trad/tref)²；openRatio = ΔP·sealRatio/pclose；expandRatio = clamp(openRatio1.15, 0, 1)。
⑥ 径向间隙关闭：ceff = c·(1−expandRatio)，Aradial = π·D·ceff·kleak·sealRatio；低压力撑不开时该泄漏保留。
⑦ 端隙更新：g = g₀ + kexpand·2πc·expandRatio；kexpand 表示理论圆周增量实际转化为开口端隙的比例，受端口斜切、止口、摩擦卡滞和局部约束影响。图示非垂直开口用 kgap 把斜切/圆角/遮挡折算为等效通流面积；贴壁后开口部位按闭合处理，Agap=0。
⑧ 高度影响：Hring 控制 sealRatio 的建立长度，同时 Agap=kgap·Hring·g·sealRatio；H 越大，端隙泄流面积越大，低进入深度起压越慢。
⑨ 总泄流面积：Atotal = Ableed + Agap + AR9 + AR6 + Aentry + Aradial + Aresidual。
⑩ 附加力：求解 Qin = Qorifice 得到稳态 ΔP；Cd 按套筒粗糙度 Rz 修正。再用 Lstop 与 βeff 形成的压缩性建压上限限制早期压差。F = sealRatio·ΔP·Aeff（MPa·mm²=N）。
⑪ 入口泄漏：Dentry 大于 Ring 外径时形成入口释放间隙，按 Aentry=π·Dentry·(Dentry−Do)/2·kleak 计入未完全密封阶段，并随 sealRatio 关闭。
⑫ 进入瞬态：进入时按 1+Ktrans·√sealRatio·e−x/Ltrans 放大瞬态压差，用于模拟密封快速建立、油液压缩和贴壁过渡造成的进入高点；随后随进入深度衰减回稳态。
⑬ Sin 采样：S 为总行程，振幅 A=S/2；θ 均匀采样，s = S/2·(1−cosθ)，v = vpeak·sinθ，261 点。
