计算过程说明
① 位移→进入深度：x = max(0, s − son)；son = 初始Ring到导向座距离 − (Ltube−Lguide)。
② 密封比：sealRatio = (x/Hring)^1.35，x=0 时 sealRatio=0 → 附加力强制为0。
③ 流量：Q = Aeff × v（m³/s）。
④ 槽面积（圆弧段）：d(x) = d₀·(1−x/L)^m，A_seg = R²·arccos[(R−d)/R] − (R−d)·√(2Rd−d²)，A_groove = N·A_seg。
⑤ Ring 胀开：pclose = Kopen·E·h·c/r²；expandRatio 与 ΔP、端隙面积自洽迭代，避免先用过高潜在压差导致 Ring 过胀。
⑥ 端隙更新：g = g₀ + kexpand·2πc·expandRatio。
⑦ 总泄流面积：Atotal = Ableed + Agap + AR9 + AR6 + Aentry + Aunseated + Aresidual。
⑧ 附加力：ΔP = ½ρ(Q/Cd/Atotal)²；F = sealRatio·ΔP·Aeff（MPa·mm²=N）。
⑨ Sin 采样：θ 均匀采样，s = S/2·(1−cosθ)，v = vpeak·sinθ，261 点。
F-S 曲线：HRS附加力 vs 复原拉伸位移（Sin振动）
