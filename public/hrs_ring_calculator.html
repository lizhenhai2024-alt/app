pclose = Kopen·E·trad·c/r²·(trad/tref)²。径向厚度按开口环截面弯曲刚度近似形成 t³ 影响；厚度越大，低压力越不容易撑开并关闭径向间隙。Hring 不直接线性乘入 pclose，但会通过 sealRatio=(x/Hring)^1.35 和 Agap=kgap·Hring·g·sealRatio 影响起压速度和端隙泄流。低压段用 ΔP·sealRatio/pclose 判断 Ring 能否撑开；撑不开时保留残余径向间隙 ceff=c·(1−expandRatio)，并按该间隙计算未贴壁泄漏。Kopen 越小，Ring 越容易在低速胀开。

尺寸敏感度： trad 主要改变 pclose（约 t³），Hring 主要改变端隙泄流面积和密封建立长度；随后再看 kgap / g0 / kleak → leakFull → Kopen → kexpand。
标定顺序： ① 用示功图实测峰值力反推总等效泄流面积 A_total； ② 实测 Ring 自由端隙 g0 代入； ③ 用低速（0.1～0.3 m/s）与高速（0.8～1.0 m/s）曲线对比，依次标定 kleak、kgap、leakFull； ④ 最后用 1.5mm 与 2.0mm 壁厚起效速度差反标定 Kopen。

计算过程说明
① 位移→进入深度：x = max(0, s − son)；son = 初始Ring到导向座距离 − (Ltube−Lguide)。
② 密封比：sealRatio = (x/Hring)^1.35，x=0 时 sealRatio=0 → 附加力强制为0。
③ 流量：Q = Aeff × v；建压有效流量 Qin = Q·sealRatio（m³/s）。
④ 槽面积（圆弧段）：d(x) = d₀·(1−x/L)^m，A_seg = R²·arccos[(R−d)/R] − (R−d)·√(2Rd−d²)，A_groove = N·A_seg。
⑤ 低压撑开：pclose = Kopen·E·trad·c/r²·(trad/tref)²；openRatio = ΔP·sealRatio/pclose；expandRatio = clamp(openRatio1.15, 0, 1)。
⑥ 径向间隙关闭：ceff = c·(1−expandRatio)，Aradial = π·D·ceff·kleak·sealRatio；低压力撑不开时该泄漏保留。
⑦ 端隙更新：g = g₀ + kexpand·2πc·expandRatio。
⑧ 高度影响：Hring 控制 sealRatio 的建立长度，同时 Agap=kgap·Hring·g·sealRatio；H 越大，端隙泄流面积越大，低进入深度起压越慢。
⑨ 总泄流面积：Atotal = Ableed + Agap + AR9 + AR6 + Aentry + Aradial + Aresidual。
⑩ 稳态附加力：求解 Qin = Qorifice + Qrg 得到 ΔP；无导向间隙泄漏时等价于 ΔP = ½ρ(Qin/Cd/Atotal)²。F = sealRatio·ΔP·Aeff（MPa·mm²=N）。
⑪ Ramp 瞬态压力：按 PDF Eq5 用 β/V·(Aeffv − Qout) 推进 HRS 腔压，隐式求解每个步长，避免已知 ramp 工况仍按 sin 速度误判峰值位置。
⑫ 导向间隙泄漏：Qrg 使用偏心环形间隙黏性流模型；当 arg=0 时关闭。
⑬ Sin 采样：仅作为对比曲线，θ 均匀采样，s = S/2·(1−cosθ)，v = vpeak·sinθ，261 点。
